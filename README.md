# 🏠 House Flex

House Flex is a colorful and interactive frontend web application designed to explore modern house designs.  
Users can view house designs, move images using the mouse, and book house design services.

This project is built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

- 🏠 House design gallery
- 🎨 Colorful user interface
- 🖱️ Drag and move house images using the mouse
- 📅 Booking form for house design services
- ⭐ Customer reviews section
- 👥 Leadership section
- ❓ Help center
- 📞 Contact information

---


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

These technologies are commonly used for building interactive web interfaces.
